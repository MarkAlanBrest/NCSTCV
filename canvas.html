Canvas Userscripts

Tampermonkey scripts for Canvas LMS. Read the install instructions below before downloading anything.

Tampermonkey is required

These are browser userscripts. You must install Tampermonkey first or none of these will work.

Install Tampermonkey for your browser

Install the Tool Dashboard first - the other scripts will not work without it

Every script on this page registers itself inside the Canvas Tool Dashboard toolbar. Without it, the other scripts will not appear anywhere.

  1. Make sure Tampermonkey is installed (see above)
  2. Download and install Canvas Tool Dashboard using the button below
  3. Then download any other scripts - they will show up in the Dashboard toolbar automatically
